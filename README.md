# Mac Wallpaper Lab (p45)

브라우저에서 맥 월페이퍼 생성 · PNG 다운로드.

**LIVE:** https://hosuman08-netizen.github.io/mac-wallpaper/

## 기능
- Mac 해상도 프리셋 (14/16 Retina, 2880×1800, 2K, FHD…)
- 스타일 6종 (Legion Gold, Aurora, Mesh, Minimal, Ocean, Ember)
- 시드·밝기 · 즐겨찾기(로컬) · 링크 공유
- PWA shell · 서버 업로드 없음 (캔버스 로컬)

## 적용
다운로드 → 시스템 설정 → 배경화면
